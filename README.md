## Commands for React
For the original reactful run (in two seperate terminals) :
```
npm run dev:bundler
npm run dev:server
```
For the npx-reactful :
```
mkdir npx-reactful
cd npx-reactful
npx reactful star-match
cd star-match

#Start the app:
npm start

#Run all tests:
npm test
```
